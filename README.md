# SMART-SUSPENSION-SYSTEM
This project presents a data-driven Smart Suspension System that predicts optimal damper settings based on road conditions and vehicle dynamics. The system uses an ARX (AutoRegressive with eXogenous inputs) model to estimate the ideal suspension response, The objective is to enhance rider's comfort, vehicle stability, and conservation of energies.

The trained ARX model predicts three optimal suspension parameters:

=>Optimal Damper Coefficient

=>Optimal Damper Force

=>Optimal Damper Displacement

The primary output is Optimal Damper Displacement, which determines how much the suspension must move to effectively absorb road impact and maintain stability.
